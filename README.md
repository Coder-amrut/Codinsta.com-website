# 🚀 Coding Platform Website

An interactive coding platform 💻 where students can learn, practice, and compete. It provides coding notes, question-solving features, group collaboration, and performance tracking in one place.

---

## 🌟 Features

### 📚 Coding Notes
- Free & paid notes for all coding topics
- Easy-to-understand structured content

### ❓ Ask Questions
- Post coding questions with:
  - Language 🧑‍💻
  - Time ⏱️
  - Description 📝

### 🏆 Leaderboard
- Global ranking system
- Points-based competition

### 👥 Group System
- Create public/private groups
- Share and solve questions
- Track who solved questions & time taken

### 📊 Dashboard
- Track your growth:
  - Questions solved ✅
  - Time spent ⏳
  - Total points 💯
  - Group performance 📈
  - Global rank 🌍

### 👤 User Profile
- Edit personal details
- Upload profile picture

### ⚙️ Admin Panel
- Add/Delete users
- Manage groups (public/private)
